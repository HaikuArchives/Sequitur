Appendix B. Links


This appendix lists other BeOS MIDI and audio applications.

B.1. MIDI Interfaces

  • MIDI Oxygen 44, a 4 in / 4 out USB-based MIDI interface.

B.2. Miscellaneous

  • MusicWeaver, complex MIDI processing.
  • Ongaku, pitch to MIDI converter.
  • Q, an analogue-style pattern generator.

B.3. Sequencers

  • dBe (formerly Devil Studio) from Softjee.
  • Dynamic Composer, an innovative addon-based tracker.
  • MeV, an open source project.

B.4. Software Synthesizers

  • 8Slots, a sample playback tool.
  • BeSting, an audio performance tool.
  • Csound, an open source synthesis tool.
  • ObjektSynth, an analogue modelling synthesizer.
