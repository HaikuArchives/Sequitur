Network Library






Most of the classes in this library are obsoleted by the new networking classes coming up in Genki, so aren't documented. However, this library does include the ArpTelnet class, which is used by ArpTelnet for its telnet remote device.

Classes

ArpTelnet (header file)

This class implements a basic part of the TELNET protocol. The class is-a BLooper, and implements the ArpRemoteTerminal BMessage communication protocol, automagically understanding about terminal sizes, emulation types, starting and stopping sessions, etc. In addition, it implements the ArpRemoteInterface, so that it can be used as an add-on in ArpRemoteAddon.
